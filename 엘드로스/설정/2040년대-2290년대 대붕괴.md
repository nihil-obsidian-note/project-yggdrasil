# 개요

대붕괴는 제4차 세계대전 직후부터 수 세기에 걸쳐 이어진 장기 문명 붕괴 시대다. 인구 감소, 국가 소멸, 오염된 생태계, 기술 단절, 기능 중심 공동체 재편이 한꺼번에 진행되며 인류는 생존 자체가 목적이 되는 시대를 통과했다.

엘드로스의 가치관과 클랜 문화, 기술 보존 집착은 모두 이 시기의 산물이다.

# 발생 배경

핵전쟁 직후 인류는 이미 산업과 행정 기반을 잃은 상태였다. 여기에 핵겨울, 방사능 오염, 식량난, 질병, 지역 분쟁이 겹치면서 구문명을 복구할 기반이 사라졌다.

# 사건의 전개

2040년대에는 핵겨울과 피폭으로 인구가 급감했다. 2050년대에는 국가 체계가 사실상 해체되며 지역 공동체 중심 생존이 일반화되었다. 2100년대에는 방사능이 일부 완화되는 대신 돌연변이 생태계와 새로운 위협이 등장했고, 2200년대 이후에는 살아남은 공동체들이 기술 편린을 이어 붙이며 제한적 재건을 시도했다.

2290년대 최후의 도시 건설은 이 긴 대붕괴기 끝에 나타난 상징적 성과였다.

# 주요 세력과 인물

이 시기의 주역은 국가나 영웅보다 생존 공동체, 기술 보유 집단, 지역 지도자, 의료와 정비 인력 같은 기능 중심 세력이었다. 살아남은 사람들은 이념보다 `무엇을 할 수 있는가`로 평가되었다.

# 직접적 결과

인류는 혈연과 국가 중심 질서를 잃고, 생존과 기술 전승을 중심으로 다시 결속했다. 안전한 물과 공기, 의료, 정비 능력, 방어력은 곧 공동체의 존속 여부를 결정하는 자산이 되었다.

# 장기적 영향

대붕괴는 이후 사회를 근본적으로 바꾸었다. 기능 공동체, 보존 우선 문화, 기술에 대한 거의 종교적인 집착, 혈연보다 역할을 중시하는 감각은 모두 이 시대의 산물이다.

# 기록과 해석

후대에서 대붕괴는 단순 공백기가 아니라 `인류가 이전 문명의 피부를 벗고 다른 생존종처럼 다시 태어난 시기`로 해석된다.

# 관련 장소와 유물

폐허 도시, 방사능 격리 구역, 반쯤 작동하는 정수 시설, 구문명 저장고, 폐기된 지하 대피소는 모두 이 시대의 유산이다.

# 현재 동향

대붕괴는 끝난 과거사이지만, 그 시대가 남긴 자원 불안과 기술 편중, 공동체 의존 감각은 지금도 엘드로스 사회 깊숙이 남아 있다.

# 시나리오 활용

플레이어는 대붕괴기의 유적 회수, 사라진 공동체 조사, 오래된 기술 기록 복원, 특정 클랜의 기원 추적 같은 방식으로 이 시대와 연결될 수 있다.
