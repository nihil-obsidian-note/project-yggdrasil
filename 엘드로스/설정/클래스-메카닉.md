# 개요

"강철의 심장이 뛰는 순간, 새로운 시대가 시작된다."

메카닉은 위그드라실의 마력과 고대 기술을 결합해 기계 장치, 의체, 자동 병기, 생체기계를 운용하는 기술형 헌터다. 전장에서의 화력뿐 아니라 장비 유지와 장기 전력 확장에서도 핵심적이다.

# 클래스의 의미

메카닉은 엘드로스가 잔존 기술과 새로운 마력 체계를 실제 군사력으로 전환하는 핵심 계통이다. 이 클래스는 기술 진보, 생존 효율, 금기의 경계를 모두 상징한다.

# 각성 계통

기술 의존형·기계 동조형 계통에 속한다. 높은 공학 이해, 정밀한 손기술, 구조 해석 감각, 위그드라실 마력과 기계 회로를 연결하는 감응성이 요구된다.

# 핵심 전투 방식

드론, 포탑, 메카, 의체, 보조 장치를 통해 원거리 화력과 방어 보조를 동시에 수행한다. 자신이 직접 싸우기도 하지만, 더 자주 전장 시스템을 설계하고 굴린다.

# 대표 능력과 기술

- 드론과 자동 포탑 운용
- 의체, 보조 장치, 생체기계 제작
- 전장 구조물 설치와 에너지 무기 사용
- 장비 긴급 수리와 전술 기계 지원

# 성장과 분화

메카닉은 크게 두 방향으로 갈린다.

## 인공기술자

인공기술자는 전장 병기와 자동화 시스템, 포탑, 드론 군단 운용에 특화된다.

## 생체기술자

생체기술자는 의체와 생체기계, 인간-기계 융합, 자율형 존재 설계에 더 깊게 파고든다.

# 포지션 구조

주 포지션은 원거리 DPS와 서포터이며, 상황에 따라 보조 탱커 역할을 겸한다. 방어 거점 구축과 지속 화력전에서 특히 강하다.

# 요구 자질과 훈련

높은 공학 이해와 자원 관리 감각, 정밀성, 안전 규율 준수가 필요하다. 훈련은 장비 설계, 정비, 위그드라실 기반 동력 운용, 기계 윤리까지 포함한다.

# 사회적 위치와 인식

메카닉은 도시 유지와 전장 기술을 동시에 책임지는 계층으로 존중받는다. 다만 신체 개조와 생체기계 연구는 경외와 불안을 동시에 부른다.

# 주요 소속과 진로

기계 정비 클랜, 후방 연구 조직, 특수 장비 개발 부대와 잘 맞는다. 고난도 장비 의존 임무와 요새전, 지원 거점 운용에서 두드러진다.

# 약점과 한계

자원 소모가 크고, 장비 의존도가 높다. 전자기 교란, 부품 부족, 통신 불능 상황에서는 전투력이 급격히 흔들릴 수 있다.

# 시나리오 활용

메카닉은 금지 개조, 자율 병기, 잃어버린 기술 복원, 카오틱 헌터의 비윤리적 개조와 맞서는 이야기에서 특히 강하게 기능한다.
