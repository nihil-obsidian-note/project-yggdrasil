# 개요

엘로이는 라의 영역 소속 B급 에스퍼다. 약 16세의 소년으로, 염동력과 공중전에 특화된 후방 전투 인원이다.

# 출신과 형성 배경

왼쪽 눈을 포함한 신체 일부를 기계로 개조한 상태다. 기계로 만든 날개와 헤일로, 신비로운 은발 때문에 `천사`라는 별명이 붙었다.

# 성격과 가치관

INFP 성향으로 기록되어 있으며, 이제 막 사춘기가 온 것 같은 츤데레 소년으로 묘사된다. 감정적이고 날것에 가까운 반응을 보이는 편이다.

# 능력과 전문성

에스퍼 클래스를 가진 B급 헌터다. 후방에서 염동력을 사용해 물체를 조작하거나 변형시키며, 공중전에 특화되어 있다.

# 소속과 역할

[[길드-라의 영역]]의 에스퍼 전력이다. 후방에서 염동력으로 전장 구조를 보조하고, 공중 기동을 통해 일반 지상 전투원이 대응하기 어려운 각도에서 작전에 기여한다.

# 대인 관계

- [[인물-다리오]]: 전담 오퍼레이터다.
- 라의 영역: `천사`라는 별명으로 인식되는 특이한 에스퍼 전력이다.

# 약점과 결핍

나이가 어리고 감정적으로 흔들리기 쉬운 면이 있다. 왼쪽 눈을 포함한 신체 일부를 기계화한 이유와 그로 인한 후유증은 확인되지 않는다.
