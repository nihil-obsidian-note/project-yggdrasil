# 개요

"모든 것은 죽음 앞에 평등하다. 하나, 죽음을 두려워말라. 그것은 언제나 우리의 곁에 있으니."

리퍼는 죽음의 기운과 영혼 조작을 전투에 활용하는 어둠 계열 헌터다. 적의 생명력 약화, 공포 유발, 처형, 언데드 운용처럼 음침하지만 강력한 수단을 통해 전장을 통제한다.

# 클래스의 의미

리퍼는 엘드로스 사회에서 가장 불길하게 여겨지면서도 가장 필요할 때 결정적인 클래스로 받아들여진다. 죽음을 다룬다는 점 때문에 공포의 대상이 되지만, 동시에 죽음의 흐름을 이해하고 통제하는 실무자로도 평가된다.

# 각성 계통

죽음 감응형·영혼 간섭형 계통에 속한다. 상실 경험, 죽음과의 근접 접촉, 강한 허무나 집요한 집념이 각성 계기가 되는 경우가 많다.

# 핵심 전투 방식

리퍼는 근접 처형, 지속 약화, 공포 유발, 언데드 보조 전술을 상황에 따라 전환한다. 단독 암살에도 강하고, 장기전에서 적을 지치게 만드는 운용도 가능하다.

# 대표 능력과 기술

- 생명력 흡수와 처형 계열 일격
- 공포, 저주, 약화 오라
- 영혼 감지와 죽음 징후 추적
- 제한적 언데드 소환과 조종

# 성장과 분화

리퍼는 크게 두 방향으로 성장한다.

## 사신

사신 계열은 직접 베고 추적하며 영혼을 수확하는 암살형 전투 계열이다. 낫과 그림자 이동, 고가치 표적 처형에 강하다.

## 리치

리치 계열은 언데드와 저주, 죽음의 영역 조성에 특화된 지배형 계열이다. 다수전과 지역 장악에서 강점을 보인다.

# 포지션 구조

주 포지션은 근접 DPS와 유틸리티다. 디버퍼, 암살자, 공포 조성자 역할을 함께 수행한다.

# 요구 자질과 훈련

죽음에 대한 공포를 견디는 정신력, 금기 지식 통제, 자기 오염 방지 훈련이 필요하다. 단순 전투력보다 심리적 균형 유지가 중요하다.

# 사회적 위치와 인식

시민 사회에서는 꺼림칙하고 불길한 존재로 보이기 쉽지만, 헌터 사회에서는 치명적 상황을 정리하는 전문가로 존중받는다. 장례와 언데드 대응 실무에 관여하는 경우도 있다.

# 주요 소속과 진로

암살, 정리, 봉쇄, 언데드 토벌, 정신 오염 지역 대응에 자주 배치된다. 특정 길드에 고정되기보다, 특수 임무 전력으로 차출되는 경우가 많다.

# 약점과 한계

신성 계열 제압과 강한 정신 방어 앞에서는 효율이 떨어질 수 있다. 죽음의 힘에 지나치게 오래 노출되면 정서 둔화와 대인 고립 위험도 커진다.

# 시나리오 활용

리퍼는 죽음의 철학, 타락 경계선, 금지된 힘의 유혹, 언데드 전쟁, 불길한 해결사 같은 이야기 축에 잘 맞는다.
