# 개요

오퍼레이터는 필드 헌터를 후방에서 지휘하고 지원하는 전략형 헌터다. 직접 적을 베는 대신, 전장의 정보와 자원, 통신, 생체 상태를 통합해 팀이 살아남고 이기게 만든다.

# 클래스의 의미

오퍼레이터는 엘드로스 전투 체계가 `개인의 강함`을 넘어 `조직적 생존`으로 발전하게 만든 핵심 계통이다. 이 클래스는 영웅보다 시스템, 일격보다 지속 승리를 대표한다.

# 각성 계통

전략 지원형·정보 운용형 계통에 속한다. 초능력 보유자일 수도 있지만, 많은 경우 능력보다 분석력과 판단력, 협업 적성이 더 중요하다.

# 핵심 전투 방식

오퍼레이터는 후방에서 전장을 읽고, 헌터의 생체 신호와 자원 상태를 분석하며, 적의 패턴과 퇴로를 계산한다. 전투력보다 전투를 가능하게 만드는 구조를 책임진다.

# 대표 능력과 기술

- 바이오 디바이스 연동과 실시간 전장 분석
- 통신, 지휘, 자원 배분, 위험 예측
- 응급 의료 지원과 상태 판단
- 팀 단위 전략 설계와 후속 대응 조정

# 성장과 분화

오퍼레이터는 크게 두 축으로 나뉜다.

## 분석형

분석형은 적 패턴, 전황 변화, 경로 선택, 판단 보조에 특화된다. 복잡한 임무일수록 가치가 커진다.

## 지원형

지원형은 장비 관리, 후방 보급, 의료 지원, 다중 팀 조율에 강하다. 장기 작전 유지력의 핵심이다.

# 포지션 구조

주 포지션은 전략 서포터다. 직접 전투보다 지휘, 통신, 정보 분석, 후방 의료에서 강점을 보인다.

# 요구 자질과 훈련

판단 속도, 멀티태스킹, 감정 통제, 장시간 집중 유지가 필요하다. 훈련은 통신망 운용, 전장 분석, 생체 신호 해석, 팀 협업을 중심으로 진행된다.

# 사회적 위치와 인식

오퍼레이터는 눈에 띄는 영웅은 아니지만, 베테랑 헌터일수록 그 중요성을 누구보다 잘 안다. 시민 사회에서는 헌터를 살려 돌려보내는 숨은 핵심 인력으로 여겨진다.

# 주요 소속과 진로

[[길드-철의 방패]]와 특히 잘 맞으며, 장거리 공략대, 구조 부대, 특수 임무 조정실에서 핵심 역할을 맡는다.

# 약점과 한계

직접 전투력은 낮은 편이며, 통신 두절이나 정보 교란에 취약하다. 잘못된 판단 하나가 팀 전체에 치명상을 줄 수 있다는 부담도 크다.

# 시나리오 활용

오퍼레이터는 보이지 않는 지휘, 후방의 죄책감, 팀 전체의 생사 판단, 분석과 감정 사이의 갈등 같은 이야기에서 빛난다.
