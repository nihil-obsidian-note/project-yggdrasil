# 개요

바이오 디바이스는 모든 헌터가 운용하는 핵심 생체 연동 장치다. 단순한 통신기나 보조 장비가 아니라, 헌터의 생존 상태와 전투 정보, 인벤토리, 클래스 정보, 오퍼레이터 연결을 한 몸처럼 묶는 실전용 인터페이스다.

# 외형과 구조

바이오 디바이스는 착용자의 신체와 밀착되는 형태로 운용되며, 팔목 장치, 신체 부착형 모듈, 보조 센서와 투사 장치를 결합한 구조로 이해된다. 외형은 장비이지만 사용 감각은 거의 신체 기관에 가깝다.

# 작동 원리

이 장치는 위그드라실의 마력을 동력으로 사용하며, 단순 에너지 공급을 넘어서 사용자의 생체 신호와 완전한 동조를 이룬다. 즉 바이오 디바이스는 기계와 마력, 생체 정보가 결합된 하이브리드 장치다.

# 핵심 기능

바이오 디바이스의 기본 기능은 통신, 생체 모니터링, 전술 정보 분석, 지도 및 적 정보 표시다. 여기에 차원 저장형 인벤토리, 장비 관리, 임무 기록 기능이 결합된다.

결과적으로 헌터는 바이오 디바이스를 통해 `보고, 기록하고, 운용하고, 연결된다`.

# 운용 조건과 사용자

주 사용자층은 헌터이며, 오퍼레이터는 연결된 바이오 디바이스를 통해 현장 정보를 읽고 판단을 지원한다. 일반 시민 장비와는 달리, 이 장치는 위그드라실 마력과 생체 동기화에 적응한 사용자를 전제로 한다.

# 클래스 관리 기능

바이오 디바이스는 헌터가 보유한 클래스와 각 클래스의 레벨, 성장 상태를 실시간으로 확인하게 해 준다. 새로운 클래스 선택 가능 시점이나 능력 운용 정보 역시 이 장치를 통해 관리된다.

# 한계와 위험

바이오 디바이스의 거의 모든 기능은 위그드라실 상태에 의존한다. 신목에 이상이 생기거나 에너지 흐름이 불안정해지면 장치 성능도 함께 흔들린다.

또한 장치가 완벽한 것은 아니기 때문에, 특정 코어나 오염 구조는 경고 체계를 속일 가능성이 있다.

# 생산과 보급

바이오 디바이스는 일반 공산품이 아니라, 정제 기술과 위그드라실 기반 자원, 정교한 제작 지식이 결합되어야 하는 전략 장비다. 따라서 보급 자체가 도시 전력 관리의 일부다.

# 사회적 의미

헌터에게 바이오 디바이스는 무기만큼 필수적인 장치이며, 시민 사회에서는 헌터라는 신분을 식별하는 상징 중 하나이기도 하다. 이 장치를 잃는다는 것은 단순 장비 손실이 아니라, 도시 전투 체계와의 연결을 잃는 것에 가깝다.
