# 엘드로스 재분류 수정 메모

## 채택한 방향

이번 수정본은 구버전 저장소의 `위그드라실` 계열 문서를 단순히 `엘드로스`로 옮기는 방식이 아니라, 새 저장소에서 이미 진행 중이던 `위그드라실 공통`과 `엘드로스 고유` 분리를 따른다.

따라서 `질서와 혼돈`, `질서와 혼돈의 우주론`은 엘드로스 단독 설정이 아니라 `위그드라실/설정`에 둔다. 반면 `헌터`, `클래스`, `정제사`, `길드`, `클랜`, `바이오 디바이스`, `신목의 요람`은 엘드로스 사회 구조에 직접 귀속되므로 `엘드로스/설정`에 둔다.

## 문장 재구성 원칙

현재 새 저장소의 `엘드로스/설정` 문서들은 골격이 좋지만 원본보다 내용이 압축되어 있었다. 이번 수정본은 새 골격을 유지하면서 구버전 원문의 세부 설명을 다시 복원하는 방식으로 작성했다.

문장 재구성은 문서의 흐름, 용어 통일, 공개/비밀 분리를 위해 수행했으며, 핵심 설정의 의미를 줄이거나 삭제하지 않는 것을 원칙으로 했다.

## 공개 문서와 비밀 문서의 분리

원본 문서 안에 HTML 주석으로 숨겨져 있던 정보는 공개 문서에서 직접 드러나지 않도록 별도 비밀 설정 문서로 분리했다. 정제사가 교감하는 존재의 정체, 계시와 영적 전쟁, 위그드라실 에너지의 정체, 몬스터 코어의 정체, 클랜 시스템의 숨은 목적, 설계자, 흑암수의 그늘이 이에 해당한다.
