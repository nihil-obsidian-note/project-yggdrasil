# 엘드로스 임시 -> 엘드로스 이행표

이 문서는 `엘드로스 임시` 문서들을 `엘드로스` 및 `위그드라실` 구조로 옮기기 전에 사용하는 분석 문서다.

현재 문서는 1차 분류표를 바탕으로, `무엇을 어느 축의 내용으로 볼 것인가`를 다시 점검하는 2차 분석 기록을 포함한다.

분류 기준은 현재까지 합의된 내용에 따른다.

- `역사` 계열은 별도 축으로 두지 않고 `사건`으로 분류한다.
- `헌터가 무엇인가`, `클래스가 무엇인가`, `길드 시스템`, `클랜 시스템` 같은 설명 문서는 `개념` 또는 `주요 개념`으로 본다.
- 개별 클래스 문서는 `클래스`로 본다.
- 공개 문서 안에 숨겨진 진실이 큰 비중을 차지하면 `비밀 설정` 분리 대상을 우선 검토한다.
- `연관문서` 메타는 이행 시 제거한다.
- 실제 이행 경로는 우선 `설정/헌터.md`, `설정/클래스.md`, `설정/위그드라실.md`처럼 `설정` 아래의 평면 문서 구조를 기준으로 본다.
- `룩스테라`와 `엘드로스`를 함께 관통하는 상위 설정은 루트의 `위그드라실` 폴더에서 관리한다.

## 분류 상태

- `유지`: 큰 구조 변경 없이 옮길 수 있음
- `분리`: 공개 정보와 비밀 정보를 나눠야 함
- `분할`: 한 문서 안에 여러 문서 축이 섞여 있어 쪼개야 함
- `보류`: 바로 옮기지 말고 참조용으로 남겨야 함
- `제외`: 이행 대상에서 제외

## 2차 분석 기준

이번 점검에서는 문서를 먼저 아래 네 축으로 판정한다.

- `위그드라실 공통`: `룩스테라`와 `엘드로스`를 함께 관통하는 상위 세계관 설정
- `엘드로스 고유`: `엘드로스`에만 직접 귀속되는 사회 제도, 장소, 인물, 장비, 사건
- `비밀 설정 우선`: 공개판보다 비밀 설정으로서의 가치가 더 큰 문서
- `혼합`: 공통 설정, 엘드로스 고유 설정, 비밀 설정이 한 문서에 섞여 있어 분할이 필요한 문서

## 현재 기준 요약

- `단체` 폴더에 있더라도 시스템 설명이면 `주요 개념`으로 재분류한다.
- `세계 규칙` 폴더에 있더라도 개별 클래스 문서면 `클래스`로 승격한다.
- `역사`는 `사건`으로 흡수한다.
- `설계자`는 공개 인물보다 `비밀 설정` 축 우선으로 본다.
- `흑암수의 그늘`은 단일 문서 유지보다 분할 전제가 맞다.
- 실제 이행 전에 먼저 `위그드라실 공통`과 `엘드로스 고유`를 가르는 내용 귀속 점검을 한 번 더 거친다.
- 실제 폴더 구조는 우선 `설정` 아래의 평면 문서 구조를 기준으로 검토한다.
