# 개요

이 클랜이 어떤 생활 기반과 정체성을 가지는지 적는다. 가족 공동체인지, 기술 전승 집단인지, 특정 구역 중심 공동체인지 먼저 분명히 적는다.

# 형성 배경

이 클랜이 어떤 붕괴와 생존 과정을 거쳐 형성되었는지 적는다. 혈연, 지역, 기술, 신념 중 무엇이 결속의 중심인지 함께 정리한다.

# 전승 기술과 생활 기반

이 클랜이 주로 보존하고 전수하는 기술, 직능, 생존 방식이 무엇인지 적는다. 무기 제작, 의료, 정비, 기록 보존 같은 핵심 기능을 분명히 적는다.

# 표식과 전통

클랜 문양, 각인 방식, 복식, 의례, 금기, 호칭 같은 정체성 표지를 적는다.

# 구성원 구조

장로, 대표자, 후계자, 교육 담당자, 일반 구성원 등 내부 구조를 적는다. 가족처럼 묶이는 방식과 실제 운영 체계가 어떻게 다른지도 적는다.

# 거주지와 활동 범위

주 거주 구역, 작업장, 보관소, 외부 활동 범위를 적는다. 도시 내부 다른 집단과 맞닿는 지점도 함께 정리한다.

# 대외 관계

다른 클랜, 길드, 정제사, 시민 집단과의 관계를 적는다. 협력과 경쟁, 의존 관계가 있다면 그 이유를 함께 적는다.

# 내부 갈등과 취약점

세대 갈등, 기술 독점 문제, 외부 압박, 전통 고수와 변화 수용의 충돌 등 내부 문제를 적는다.

# 현재 동향

현재 이 클랜이 지키려는 것, 잃어가고 있는 것, 새롭게 얻으려는 것을 적는다.

# 시나리오 활용

플레이어가 이 클랜과 어떤 관계를 맺을 수 있는지 적는다. 보호, 거래, 기술 전수, 갈등 중재, 비밀 탐색 같은 접점을 중심으로 정리한다.
