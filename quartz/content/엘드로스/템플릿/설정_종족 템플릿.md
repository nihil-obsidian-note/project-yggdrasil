# 개요

이 종족이 어떤 존재이며 엘드로스 세계에서 어떤 위치를 차지하는지 적는다. 인간계 종족인지, 변이 종인지, 게이트 유래 종족인지 먼저 분명히 적는다.

# 기원

이 종족이 어디에서 비롯되었는지 적는다. 자연 진화, 게이트 기원, 오염 변이, 위그드라실 영향 같은 배경을 함께 정리한다.

# 외형과 신체적 특징

평균 체격, 피부나 기관의 특징, 감각 능력, 수명, 생리적 특성을 적는다.

# 생활 방식과 문화

주거 형태, 가족 구조, 가치관, 의례, 금기, 교육 방식, 생존 전략을 적는다.

# 사회 구조

지도층, 공동체 단위, 계급 구조, 의사결정 방식 같은 사회 내부 구조를 적는다.

# 능력과 한계

이 종족이 가진 고유한 재능, 적응력, 약점, 질병 취약성, 환경 의존성을 적는다.

# 타 세력과의 관계

인간, 길드, 클랜, 정제사, 다른 종족과의 관계를 적는다. 공존, 갈등, 배척, 동맹 양상을 함께 정리한다.

# 주요 거주지

이 종족이 어디에 주로 정착하거나 출몰하는지 적는다. 도시 내부, 외곽, 게이트 너머, 특정 환경권과의 연관을 함께 적는다.

# 현재 동향

증가, 쇠퇴, 동화, 분열, 추방, 정치 세력화 같은 현재 변화를 적는다.

# 시나리오 활용

플레이어가 이 종족과 어떤 방식으로 엮일 수 있는지 적는다. 협상, 충돌, 보호, 연구, 공존 문제 같은 접점을 중심으로 정리한다.
