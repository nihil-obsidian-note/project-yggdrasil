# 개요

클랜은 대붕괴 이후 혈연과 기존 가계 체계가 붕괴한 인류가 새롭게 만들어 낸 생활 공동체 시스템이다. 길드가 전투와 임무 중심 공동체라면, 클랜은 생활, 기술 전승, 생존 윤리, 정체성의 기반을 담당하는 집단이다.

엘드로스 시민 다수에게 클랜은 단순한 조직이 아니라, 실제 가족을 대신하는 가장 가까운 사회 단위다.

# 형성 배경

대붕괴 이후 생존자들은 혈연보다 기능과 생존 경험을 중심으로 결속했다. 같은 지역에서 살아남은 사람들, 같은 기술을 지닌 사람들, 같은 방식으로 버텨 온 사람들이 느슨한 생존 단위를 이루었고, 시간이 지나며 이것이 클랜으로 발전했다.

이 과정에서 클랜은 피의 연속성보다 기억과 기술, 의례와 상호 부양을 이어 주는 단위가 되었다.

# 전승 기술과 생활 기반

클랜은 각자 고유한 생활 기반과 기술 전통을 가진다. 어떤 클랜은 무기 제작에, 어떤 클랜은 의료와 생명과학에, 어떤 클랜은 기계 정비와 개조에, 어떤 클랜은 기록 보존과 연구에 강점을 보인다.

따라서 클랜은 단순 거주 공동체가 아니라, 도시의 일상과 장기 생존을 떠받치는 분산형 기술 저장소이기도 하다.

# 표식과 전통

클랜의 이름과 문양은 몸 어딘가의 각인이나 의복, 장신구로 드러나는 경우가 많다. 위그드라실 수액을 이용한 특수 각인, 색상, 장식, 의례는 클랜의 역사와 철학, 성취를 표현하는 수단이다.

개인에게 클랜 표식은 소속 증명 이상의 의미를 갖는다. 그것은 자신이 누구의 보호를 받고, 누구의 기억을 이어받았는지 보여 주는 표지다.

# 구성원 구조

클랜은 장로, 대표자, 기술 전수자, 실무 구성원, 후계 세대로 구성되는 경우가 많다. 혈연 공동체처럼 보이지만 실제 운영은 기술 전수와 생존 협력이 핵심인 기능 공동체에 가깝다.

이 때문에 클랜은 따뜻한 가족성과 냉정한 생존 기능을 동시에 가진다.

# 주요 클랜

현재 대표적으로 거론되는 클랜에는 철혈 클랜, 푸른 나선 클랜, 황금 톱니 클랜, 은빛 서책 클랜이 있다. 각각 군사 기술, 의료·생명과학, 기계 정비, 기록과 연구처럼 도시 생존의 서로 다른 기반을 담당한다.

# 대외 관계

클랜은 길드, 정제사, 도시 행정과 긴밀하게 연결되어 있다. 길드가 전장에 나서는 동안, 클랜은 장비를 고치고, 사람을 치료하고, 지식을 보존하고, 일상을 유지한다.

그러나 이런 중요성 때문에 특정 기술 독점이나 자원 배분 문제를 둘러싼 긴장도 발생한다.

# 현재 동향

엘드로스가 더 복잡한 도시가 될수록 클랜은 단순한 생활 공동체를 넘어 정치적·기술적 이해관계의 중심으로 이동하고 있다. 전통을 지키려는 흐름과 외부와 더 적극적으로 결합하려는 흐름 사이의 긴장도 커지고 있다.
