# 개요

미우는 라의 영역 소속 A급 메카닉이다. 약 22세의 헌터로, 후방에서 중화기를 이용해 폭발적인 화력을 투사한다.

# 출신과 형성 배경

어릴 때부터 쉽게 병들고 약한 몸을 가지고 있었다. 강해지기 위해 의식을 제외한 모든 신체를 기계화했으며, 고양이 귀를 단 것은 순전히 그녀의 취향으로 기록되어 있다.

# 성격과 가치관

ENFP 성향으로 기록되어 있으며, 명랑하고 말이 많다. 표정에 감정이 강하게 드러나는 편은 아니지만, 단순하고 직관적인 성격을 지닌다.

# 능력과 전문성

메카닉 클래스를 가진 A급 헌터다. 중화기를 주무기로 사용하며, 후방에서 순간 화력을 집중해 적을 제압한다. 전신 기계화 상태이기 때문에 일반적인 인간형 헌터와 다른 내구성과 운용 방식을 가진다.

# 소속과 역할

[[길드-라의 영역]]의 후방 화력 담당이다. 전방 돌파조가 적의 시선을 붙잡는 동안, 중화기로 고화력 딜링을 수행한다.

# 대인 관계

- [[인물-카야]]: 전담 오퍼레이터이며, 함께 전투에 참여해 바이오 디바이스 역할을 대신한다.
- 라의 영역: 후방 화력축을 담당한다.

# 약점과 결핍

의식을 제외한 모든 신체를 기계화한 상태다. 기계화 이전의 약한 몸, 전신 기계화가 남긴 정체성 문제, 유지보수 의존성은 약점으로 사용할 수 있으나 상세한 사건은 확인되지 않는다.

# 현재 동향

라의 영역에서 중화기 기반 후방 화력 담당으로 활동한다.

# 시나리오 활용

미우는 전신 기계화 헌터의 인간성, 약한 몸에서 강해지려 한 선택, 카야와의 전투 페어 관계를 중심으로 활용할 수 있다. 장비 고장, 신체 유지보수, 기계화 윤리 문제와 엮기 좋다.
