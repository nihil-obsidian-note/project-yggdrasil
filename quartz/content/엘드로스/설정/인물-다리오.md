# 개요

다리오는 라의 영역 소속 오퍼레이터다. 약 35세의 인물로, 엘로이의 전담 오퍼레이터를 맡고 있다.

# 출신과 형성 배경

라의 영역에서 가장 활발하게 대외 활동을 하며 외부 정보를 수집한다. 출신, 합류 배경, 과거 이력은 확인되지 않는다.

# 성격과 가치관

ENTJ 성향으로 기록되어 있으며, 능글맞고 사교적인 성격이다. 호감형의 인상을 가지고 있으나, 길드 안에서는 가끔 무슨 생각을 하는지 알 수 없다는 평을 듣는다.

# 능력과 전문성

사람들의 반응과 분위기를 잘 파악하며, 교섭에 최적화된 언행을 구사한다. 외부 정보를 수집하는 능력이 강하고, 은밀한 정보를 케프리에게 보고한다.

# 소속과 역할

[[길드-라의 영역]]의 오퍼레이터이며, [[인물-엘로이]]의 전담 지원 담당이다. 동시에 길드의 대외 활동과 정보 수집을 맡는 인물이다.

# 대인 관계

- [[인물-엘로이]]: 전담 지원 대상이다.
- [[인물-케프리]]: 수집한 은밀한 정보를 보고한다. 다만 모든 정보가 케프리에게 보고되는지는 확인되지 않는다.
- 라의 영역: 가장 비밀이 많을 것으로 유추되는 인물이다.

# 약점과 결핍

비밀이 많다는 평판 자체가 가장 큰 약점이다. 정보 일부를 숨기고 있을 가능성이 있으나, 실제로 무엇을 감추는지는 확인되지 않는다.

# 현재 동향

엘로이 전담 오퍼레이터로 활동하면서, 라의 영역의 외부 정보망과 교섭 창구 역할을 수행한다.

# 시나리오 활용

다리오는 라의 영역 내부의 불완전한 정보 흐름, 은밀한 외교, 배신 가능성, 이중 정보원 서사에 적합하다. 플레이어에게 정보를 제공하면서도 모든 것을 말하지 않는 조력자로 사용할 수 있다.
